# 추상팩토리 패턴

* 특정 범주로 묶이는 개념들을 하나의 단위로 추상화 하여 제공

* 장점
  * 기능의 수평적 확장에 용이하다
* 특징
  * 기능의 수직적 확장이 필요한 경우 추상 팩토리를 확장한 확장된 인터페이스 구현

### 예시 기능 - 커피 머신
* 커피 머신 (Interface)
  * A 커피 머신
    *  A 아메리카노 추출
  * B 커피 머신
    * B 아메리카노 추출
  * C 커피 머신
    * C 아메리카노 추출
  * D 커피 머신
    * D 아메리카노 추출
    * D 라떼 추출 (커피 머신 화장)

* 라뗴 머신 (커피 머신 확장 interface)
* 아메리카노 (interface)
* 라떼 (interface)
